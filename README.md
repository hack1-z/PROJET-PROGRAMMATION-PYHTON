# PROJET-PROGRAMMATION-PYHTON
Bases de la Programmation en Python

## 🐍 Apprendre Python : De la Théorie à la Pratique
Bienvenue dans ce dépôt dédié à l'apprentissage de la programmation Python. Ce projet est conçu pour accompagner les étudiants à travers un parcours structuré : des concepts fondamentaux aux projets concrets.
------------------------------
## 🎯 Objectifs du Programme

* Maîtriser la syntaxe de base et les structures de données.
* Comprendre la logique algorithmique et la Programmation Orientée Objet (POO).
* Appliquer les connaissances via des ateliers pratiques.
* Collaborer sur GitHub en utilisant les bonnes pratiques de développement (Git, Clean Code).

------------------------------
## 📂 Organisation du Dépôt
Le contenu est divisé en trois piliers principaux pour faciliter la navigation :

* 📚 /cours : Guides textuels et Notebooks Jupyter détaillant chaque concept (Variables, Boucles, Fonctions, Bibliothèques).
* 🖼️ /slides : Supports visuels (PDF/Markdown) utilisés lors des présentations en séance.
* 🛠️ /projets : Études de cas et exercices pour valider la compréhension (ex: Calculatrice, Analyse de données, Web Scraping).

------------------------------
## 🛠️ Configuration & Outils Recommandés
Pour une expérience de développement optimale, voici l'environnement de travail préconisé :
## 1. Python 🐍
Installez la version Python 3.10 ou supérieure.

* Lien : python.org/downloads
* Note : Sous Windows, cochez bien "Add Python to PATH" lors de l'installation.

## 2. Éditeur de Code (IDE) 💻

* Visual Studio Code (Recommandé) : Installez l'extension "Python" de Microsoft. C'est l'outil le plus polyvalent.
* PyCharm Community : Une alternative puissante pour les projets plus complexes.
* Python 3.X : est facile à apprendre, que vous soyez un programmeur débutant ou expérimenté dans d'autres langages. Les pages suivantes constituent un premier pas utile pour vous lancer dans la programmation en Python !

## 3. Gestion de projet & Git 🐙

* Git : Indispensable pour versionner votre code. Installer Git.
* Environnement Virtuel : Apprenez à isoler vos projets pour éviter les conflits :

python -m venv venv# Activer 
(Windows): .\venv\Scripts\activate# Activer 
(Mac/Linux): source venv/bin/activate


------------------------------
## 🚀 Comment démarrer ?

   1. Cloner le projet :
   
   git clone https://github.com
   cd nom-du-projet
   
   2. Installer les dépendances :
   
   pip install -r requirements.txt
   
   3. Explorer : Commencez par le dossier /cours pour les bases ou /projets pour tester vos compétences.
